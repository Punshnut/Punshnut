**Hi, I’m Jan 👋**

I build intentional software and visual work — tools that reduce noise and respect attention.

For over a decade I've worked across disciplines: low-level systems (C, C#, Java), mobile (Android & iOS), networking, automation and robotics. Today I focus primarily on native macOS utilities (Swift, SwiftUI, AppKit), while moving across Windows, Linux, mobile and web when the problem calls for it. I choose platforms deliberately, guided by the work and free from ecosystem loyalty.

Good software doesn't demand attention. It supports focus, reduces friction, and quietly does its job. No analytics, telemetry or tracking. Apps don't phone home. Updates are optional via Sparkle (GitHub releases), and network access can be fully blocked without affecting functionality. AI code completion is part of my workflow while architecture, structure and direction remain mine.

Alongside software, I work as a freelance filmmaker and photographer, guided by kindness, creativity and inclusion. Time spent outdoors across different climates and landscapes shapes how I think about systems: resilient, adaptive, minimal and intentional.

**Clarity over noise. Depth over hype. Intention over volume.**
