Hi, I’m Jan 👋

I build intentional software, systems and visual work - tools that reduce noise and respect attention.

I’m 27 and have been building across disciplines for over a decade. My background spans low-level programming (C, C#, Java), mobile development (Android & iOS), networking, automation and robotics. I enjoy working across the stack - from system-level logic to interface design - and thinking deeply about system behavior.

Today, I build primarily native macOS utilities (Swift, SwiftUI, AppKit), while also working across Windows, Linux, mobile and web environments. I choose platforms deliberately - based on the problem, not the ecosystem.

Good software doesn’t demand attention. It supports focus, reduces friction, and quietly does its job.

I don’t include analytics, telemetry or tracking, and apps don’t phone home. Updates are optional via Sparkle (GitHub releases), and you can fully block network access without affecting functionality.

AI-assisted coding is part of my workflow - a collaborative tool that helps maintain momentum while I remain responsible for structure and direction.

Alongside software, I work as a freelance filmmaker and photographer - guided by kindness, creativity and inclusion. I spend much of my time outdoors across different climates and landscapes, and that connection to nature shapes how I think about systems: resilient, adaptive, minimal and intentional.

Whether I’m building cross-platform software, automation systems, robots or visual work, the principle stays the same:

Clarity over noise.<br />
Depth over hype.<br />
Intention over volume.<br />
