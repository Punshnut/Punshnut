**Hi, I’m Jan 👋**

I build intentional software and visual work - tools that reduce noise and respect attention.

For over a decade, I’ve worked across disciplines: low-level systems (C, C#, Java), mobile (Android & iOS), networking, automation and robotics. I enjoy moving across the stack - from system logic to interface design - thinking carefully about behavior, structure and long-term maintainability.

Today I focus primarily on native macOS utilities (Swift, SwiftUI, AppKit), while also working across Windows, Linux, mobile and web when the problem calls for it. I choose platforms deliberately - based on the problem, not the ecosystem.

Good software doesn’t demand attention. It supports focus, reduces friction, and quietly does its job.

I don’t include analytics, telemetry or tracking. Apps don’t phone home. Updates are optional via Sparkle (GitHub releases), and network access can be fully blocked without affecting functionality.

AI code completion is part of my workflow - I remain responsible for architecture, structure and direction.

Alongside software, I work as a freelance filmmaker and photographer - guided by kindness, creativity and inclusion. I spend much of my time outdoors across different climates and landscapes, and that connection to nature shapes how I think about systems: resilient, adaptive, minimal and intentional.
